Project can be run by executing run_CSERF.py

"Algosuit+Middlebury" Dataset is available from https://github.com/ahmadh84/occlusiontracking/blob/master/download_dataset.m

The features were generated from Humayun et al's Occlusions pipeline available from http://visual.cs.ucl.ac.uk/pubs/learningOcclusion/

These generated features are 25GB when compressed, which is too large for uploading onto OneDrive
